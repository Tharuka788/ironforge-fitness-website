# IronForge Fitness - Promotional Landing Page

A premium, high-performance landing page developed for a local fitness brand as part of the **Koncepthive Web Development Internship Evaluation**. This project demonstrates a high-fidelity "Figma-to-Code" implementation with a focus on modern UI/UX principles and functional interactivity.

## ✨ Key Features
* **Mobile-First Responsiveness:** Fully optimized for Mobile, Tablet, and Desktop screens using Tailwind CSS media queries.
* **Dynamic Service Filtering (Bonus):** Interactive filtering system built with React state, allowing users to sort gym classes by categories: **Strength, Cardio, and Wellness**.
* **API Integration (Bonus):** Integrated with **Web3Forms API** for real-time contact form handling and automated email notifications.
* **Advanced Form Handling:** Includes field validation for Name, Email (pattern matching), and Message, with visual success feedback upon submission.
* **Minimalist Dark Mode:** A sleek aesthetic featuring a Gold and Dark-grey color palette to match the premium fitness brand identity.

---

## 🛠️ Tech Stack
* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **API Handling:** Web3Forms API
* **Icons:** Lucide-React / FontAwesome
* **Deployment:** Vercel
